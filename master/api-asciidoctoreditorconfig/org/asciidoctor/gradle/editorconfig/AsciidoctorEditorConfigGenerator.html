Package: org.asciidoctor.gradle.editorconfig

[Groovy] Class AsciidoctorEditorConfigGenerator

        • org.asciidoctor.gradle.editorconfig.AsciidoctorEditorConfigGenerator 

        • @groovy.transform.CompileStatic
class AsciidoctorEditorConfigGenerator
extends DefaultTask

          Generates .asciidoctorconfig file. When the file is generated attributes are applied in the following order.

          1. Directly specified attributes
          2. Attributes obtaining via org.asciidoctor.gradle.base.AsciidoctorAttributeProvider instances
          3. Appending content of files which were specified as additional providers.

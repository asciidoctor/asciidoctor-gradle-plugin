              The document conversion might generate additional artifacts that could require copying to the final destination. An example is use of ditaa diagram blocks. These artifacts can be specified in this block. Use of the option implies useIntermediateWorkDir. If copyNoResources is set or copyResourcesOnlyIf(String ...) does not match the backend, no copy will occur.

              Parameters:
              cfg - Configures a PatternSet with a base directory of the intermediate working directory.

          • @Override void setExecutionMode(org.ysb33r.grolifant.api.core.jvm.ExecutionMode em)

            Sets the execution mode.

            This allows for JVM tasks to be executed either on a worker queue inside the JVM using an isolated classpath, ouside the JVM in a separate process, OR using a classic javaexec.

            If nothing is set, the default is org.ysb33r.grolifant.api.core.jvm.ExecutionMode#JAVA_EXEC.

            Parameters:
            em - Execution mode.
            Throws:
            UnsupportedConfigurationException is mode is illegal within context.
